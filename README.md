==== Q-Express GraPARI Bot ====

Q-Express GraPARI Bot adalah bot Telegram untuk mengelola sistem antrian layanan GraPARI secara sederhana menggunakan Google Sheets sebagai database.

Bot berjalan menggunakan Node.js, Telegram Bot API, serta dapat di-deploy dengan mudah menggunakan Railway.

A. Fitur :

- Ambil nomor antrian secara otomatis

- Menyimpan data antrian ke Google Sheets

- Melihat nomor antrian terakhir

- Deployment cloud menggunakan Railway

- Sistem ringan tanpa database server

B. Teknologi yang Digunakan :

- Node.js

- Telegram Bot API

- Google Sheets

- Railway

C. Lisensi 
  
  Project ini bebas digunakan untuk kebutuhan pengembangan dan pembelajaran.
